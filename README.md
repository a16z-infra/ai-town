## Installation

Client:

```bash
cd game
npm install       # install the dependencies
npm run build     # bundling up resources
npm start         # start server
```

Server:

```bash
npm install
npm run dev
```

You can now visit http://localhost:[PORT_NUMBER]/ai-town
